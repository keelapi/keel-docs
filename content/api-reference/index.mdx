This section is intentionally a placeholder until the OpenAPI-backed reference is integrated into this Nextra site.

- a FastAPI-generated OpenAPI schema in non-production environments
- authored documentation in the `../keel-api/docs` source tree
- public runtime routes under `https://api.keelapi.com`
