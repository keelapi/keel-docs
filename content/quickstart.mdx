# Quickstart

Send your first governed request in under 60 seconds.

## Step 0: Set up your project

Open the [Keel dashboard](https://dashboard.keelapi.com).

Do this first:

1. Create a project.
2. Copy the project API key.
3. Add a provider key to that same project.

## Step 1: Set your API key

```bash
export KEEL_API_KEY="keel_sk_live_demo_123456789"
```

## Step 2: Run one request

```bash
curl -sS https://api.keelapi.com/v1/executions \
  -H "Authorization: Bearer $KEEL_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: quickstart-exec-001" \
  -d '{
    "operation": "generate.text",
    "messages": [
      {
        "role": "user",
        "content": "Reply with exactly: keel works"
      }
    ],
    "routing": {
      "provider": "openai",
      "model": "gpt-4o-mini"
    },
    "parameters": {
      "max_output_tokens": 16,
      "temperature": 0
    }
  }'
```

## What just happened

- Keel authenticated your project with the API key.
- Keel evaluated a permit internally for this request.
- Keel routed the request to the configured provider and model.
- Keel returned one normalized response envelope with output, routing, governance, and usage.
